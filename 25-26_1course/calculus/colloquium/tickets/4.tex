\section{Непрерывность функции в точке, на отрезке, на интервале. Область
непрерывности функций: степенной (для натурального показателя),
тригонометрических. (Beta)}

\subsection{Непрерывность функции в точке}

\begin{definition}
Функция $f$ называется непрерывной в точке $x_0$, если:
\begin{enumerate}
\item $f(x_0)$ определена;
\item существует предел $\Lim{x}{x_0} f(x)$;
\item $\Lim{x}{x_0} f(x)=f(x_0)$.
\end{enumerate}
\end{definition}

Эквивалентная запись:
\[
\Lim{x}{x_0}\bigl(f(x)-f(x_0)\bigr)=0.
\]

В $\varepsilon$–$\delta$ форме:
\[
\forall \varepsilon>0\ \exists \delta>0:
|x-x_0|<\delta \Rightarrow |f(x)-f(x_0)|<\varepsilon.
\]

\subsection{Непрерывность на множестве}

\begin{definition}
Функция называется непрерывной на множестве $E$, если она непрерывна в каждой точке этого множества.
\end{definition}

Для отрезка $[a,b]$:
\begin{itemize}
\item в точках $(a,b)$ — обычная непрерывность;
\item в точке $a$ — непрерывность справа;
\item в точке $b$ — непрерывность слева.
\end{itemize}

\subsection{Связь с пределом}

Функция непрерывна в точке $x_0$ тогда и только тогда, когда
\[
\Lim{x}{x_0} f(x)=f(x_0).
\]

Если функция дифференцируема в точке, то она непрерывна в этой точке.

\subsection{Область непрерывности степенной функции (натуральный показатель)}

Рассмотрим функцию
\[
f(x)=x^n, \qquad n\in\mathbb{N}.
\]

\begin{theorem}
Функция $x^n$ непрерывна на всей числовой прямой $\mathbb{R}$.
\end{theorem}

\begin{Proof}
При $n=1$ очевидно.
Если $n>1$, то
\[
x^n = x\cdot x^{n-1}.
\]
Так как произведение непрерывных функций непрерывно,
а функция $x$ непрерывна,
то по индукции получаем непрерывность $x^n$ на $\mathbb{R}$.
\end{Proof}

Следовательно, любой многочлен непрерывен на $\mathbb{R}$.

\subsection{Область непрерывности тригонометрических функций}

Из первого замечательного предела следует непрерывность функции $\sin x$ в нуле:
\[
\Lim{x}{0}\sin x = 0.
\]

Используя формулы сложения, получаем непрерывность $\sin x$ в любой точке.

\begin{theorem}
Функции
\[
\sin x,\quad \cos x
\]
непрерывны на всей $\mathbb{R}$.
\end{theorem}

Функции
\[
\tan x=\frac{\sin x}{\cos x}, \qquad
\cot x=\frac{\cos x}{\sin x}
\]
непрерывны во всех точках своей области определения:

\[
\tan x \text{ непрерывна при } x\neq \frac{\pi}{2}+k\pi,
\]
\[
\cot x \text{ непрерывна при } x\neq k\pi,
\quad k\in\mathbb{Z}.
\]

\subsection{Примеры}

\begin{example}
Функция
\[
f(x)=x^3+2x+1
\]
непрерывна на всей $\mathbb{R}$ как многочлен.
\end{example}

\begin{example}
Функция
\[
f(x)=\tan x
\]
непрерывна на каждом интервале
\[
\left(-\frac{\pi}{2}+k\pi,\ \frac{\pi}{2}+k\pi\right).
\]
\end{example}
